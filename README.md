# VoltPocket-Portable-USB-Power-Module
A compact portable power module that converts regulated input voltage into a stable 5V USB output for charging smartphones and small electronic devices.
Overview
This project implements a basic voltage regulator circuit that converts 220V AC mains supply into a regulated 5V DC output. The output can be used for mobile phone charging (prototype) and as a power source for low-voltage electronic circuits such as ICs, microcontrollers, and breadboard projects.

The project focuses on applying fundamental analog electronics concepts in a simple and practical manner.

# Objective
Convert high-voltage AC to low-voltage DC
Understand rectification, filtration, and voltage regulation
Design a stable 5V regulated power supply
Build a useful real-life electronics prototype
# Circuit Description
The circuit consists of four main stages:

1.Step-Down Transformer – Reduces 220V AC to low-voltage AC
2.Bridge Rectifier – Converts AC to pulsating DC
3.Filter Capacitor – Reduces ripple in the DC output
4.7805 Voltage Regulator – Provides a constant 5V DC output
# Components Used
1.9-0-9V Step-Down Transformer
2.PN Junction Diodes (×4)
3.Capacitor (1000 µF)
4.7805 Voltage Regulator IC
5.Breadboard and Connecting Wires
# Output
1.Output Voltage: 5V DC
2.Maximum Current: ~1A
# Applications
Mobile phone charger (educational prototype)
Power supply for digital and analog circuits
Breadboard and lab experiments
